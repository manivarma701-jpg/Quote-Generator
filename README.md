# Quote-Generator

A simple and elegant web application that generates random inspirational quotes with a clean, user-friendly interface.

## Features

- 🎯 **Random Quote Generation** - Get inspired with random quotes at the click of a button
- 🎨 **Beautiful UI** - Modern and responsive design for all devices
- 📱 **Mobile Friendly** - Optimized for desktop and mobile viewing
- ⚡ **Fast & Lightweight** - Quick loading and minimal dependencies
- 💾 **Easy to Use** - Simple interface with intuitive controls

## Technology Stack

- **JavaScript** (53%) - Core functionality and interactivity
- **CSS** (34.7%) - Styling and responsive design
- **HTML** (12.3%) - Page structure and markup

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/manivarma701-jpg/Quote-Generator.git
cd Quote-Generator
```

2. Open the project:
```bash
# Simply open index.html in your web browser
open index.html
```

Or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (http-server)
npx http-server
```

3. Visit `http://localhost:8000` in your browser

## Usage

1. Open the application in your web browser
2. Click the "Generate Quote" button to display a random quote
3. Share or copy your favorite quotes
4. Generate as many quotes as you'd like!
